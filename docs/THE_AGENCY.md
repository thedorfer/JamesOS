# The Agency

Last reviewed: 2026-07-18

The Agency is a permanent main-interface anchor for understanding and supervising JamesOS agents. The recovery branch provides a deterministic placeholder view; the operational workspace is planned.

Agents may return bounded text, registered view suggestions, local form patches, placement hints, diagnostics, and confirmation requests. They may not generate executable HTML/JavaScript, remove or reorder Context Dock anchors, overwrite persisted layouts, remove system locks, submit forms, or perform external writes beyond declared capabilities and approval policy.

Jade/system policy has final authority over navigation anchors, components, locks, destinations, approvals, terminal access, privileges, and safe failure. Optional capabilities must report unavailable state without destabilizing other workspaces.

The planned Agency workspace shows active agents, current runs, assignments, pending approvals, recent results, tools/capabilities, and Jade safety decisions. Every agent declares inputs, outputs, local/network/external effects, provider access, terminal/privilege requirements, and rollback behavior.

[The Marine](THE_MARINE.md) is a special optional easter-egg agent. It is planned as a capability-free sandboxed game workspace, not a terminal or provider agent.

See [Building agents](BUILDING_AGENTS.md) and [Workspace Engine](WORKSPACE_ENGINE.md).
