# Creative Foundations

Creative Foundations are read-only planning libraries for future image and product work.

Storage:

```text
~/JamesOSData/JamesOS/CreativeStudio/PromptLibrary/
~/JamesOSData/JamesOS/CreativeStudio/Assets/
~/JamesOSData/JamesOS/CreativeStudio/Styles/
~/JamesOSData/JamesOS/Reports/Creative Foundations.md
```

## Prompt Library

Includes local templates for positive prompts, negative prompts, typography, transparent PNG, product art, and mockups.

## Asset Library

Scans metadata for SVG, PNG, JPG, WEBP, TTF/OTF, AI, EPS, and PDF files.

Font files are metadata-only. Asset records do not include file contents.

## Style Registry

Default styles:

- typography
- retro
- minimalist
- cute
- bold
- watercolor
- vintage
- sticker
- pride
- Thai/English

## Safety

These libraries only enrich image plans. They do not execute ComfyUI, generate images, call Printify, call Etsy, upload, publish, order, or send anything.
