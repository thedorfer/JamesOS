# Contributing to JamesOS

Thank you for your interest in JamesOS.

Issues, bug reports, testing feedback, documentation suggestions, and
architecture discussions are welcome.

JamesOS is licensed under the PolyForm Noncommercial License 1.0.0.

Code contributions are not currently accepted for merging unless the
contributor has entered into a separate contributor agreement with the
project owner. This is necessary to preserve the project owner's ability
to use and license JamesOS commercially.

Do not include credentials, private deployment information, account IDs,
shop names, product IDs, listing IDs, private artwork, or personal data
in issues or pull requests.
